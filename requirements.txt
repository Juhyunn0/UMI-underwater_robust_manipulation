# =============================================================================
# UMI Underwater Robust Control — unified dependency manifest
#
# This repo bundles two subprojects that share one environment:
#   1. Refractive TagSLAM + gantry  (src/)                  — gtsam, pyzed, opencv
#   2. BlueROV2 DOBMPC MuJoCo sim   (bluerov2_mujoco_dobmpc/) — mujoco, casadi
#
# NumPy is pinned <2.0 on purpose: gtsam 4.2.x is built against the NumPy 1.x
# ABI and *requires* numpy<2.0.0. pyzed 5.2's wheel metadata declares
# numpy>=2.0, but that is a conservative build-time pin — pyzed.sl imports and
# runs correctly on 1.26.4 (verified). So `pip check` will print a benign
# "pyzed ... requires numpy<3.0,>=2.0 but you have 1.26.4" warning; ignore it.
# Do NOT bump numpy to 2.x — it breaks the gtsam SLAM backend.
# For an exact, reproducible pin of the whole stack, use requirements-lock.txt.
# =============================================================================

# Core scientific stack
