# FMC4030 Return Codes

| Code | Meaning         | Typical Action |
|------|-----------------|----------------|
| 0    | 执行成功        | 无             |
| -1   | 连接失败        | 检查网线、IP、端口、重启控制器 |
| -2   | 保留（未使用）  | — |
| -3   | 保留（未使用）  | — |
| -4   | 数据建立失败    | 检查内存、参数 |
| -5   | 数据发送失败    | 检查网络连接（线缆/IP/端口/重启控制器） |
| -6   | 数据接收失败    | 检查网络连接（线缆/IP/端口/重启控制器） |
| -7   | 接收数据错误    | 检查网络连接 |
| -8   | 空指针错误      | 确认传入指针有效 |
